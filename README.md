### Commit 1 Reflection Notes
---
Di Milestone 1, saya belajar membangun sebuah *web server single-threaded* sederhana menggunakan Rust. Langkah pertama yang dilakukan adalah membuat *server* mendengarkan koneksi TCP pada *socket* lokal di alamat `127.0.0.1:7878` melalui komponen `TcpListener`. Selanjutnya, mengimplementasikan `handle_connection` untuk menerima dan memproses *stream* data dari setiap koneksi TCP yang berhasil tersambung. Proses membaca permintaan HTTP dari *browser* ini dioptimalkan  `BufReader` agar arus data dapat dibaca secara terstruktur baris demi baris menggunakan `.lines()`. *Raw data* yang masuk tersebut kemudian diambil dan dikumpulkan ke dalam sebuah *Vector* hingga program mendeteksi adanya baris kosong penanda akhir permintaan. Melalui tahapan ini, saya dapat melihat langsung struktur komunikasi yang dikirimkan oleh *client*, seperti metode `GET` dan rincian *header* pendukung lainnya, sesaat sebelum *server* memberikan balasan.
